x4-diagrams
===========

Diagrams illustrating [X4] faction relationships and economy flows.

This project is entirely unaffiliated with Egosoft.

Usage
-----

```shell
python -m x4 --help
python -m x4.factions
python -m x4.economy
```

Graphviz `dot` files will be written to an `output/` directory.
